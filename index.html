中级社会工作实务 · 真题精讲学习笔记

基于小梦老师直播课 · 2021年真题做题方法与核心考点

一、答题总纲（考前必看）

中级实务没有标准答案，言之有理即可得分！

四个核心共识：

  • 不需要和教材小标题写得一模一样
  • 案例中提到的内容不用全抄，点到即可
  • 扩写部分是锦上添花，不是必须一字不差
  • 答案和参考要点不一样 ≠ 没有分数

做题顺序：从简单到难，控制好时间！

先做最有把握的题目拿满分，最难的题最后做，实在不会也要写——写了就有分，终极实务不存在零分。

分数策略 1:2:1 → 简单题冲13-14分 | 中等题拿8-9分 | 难题拿5分保底

万能答题结构 = 小标题 + 案例中提到 + 一句扩写

扩写不会写？用"表明了……的需求"或"说明了……的问题"来衔接。

二、案例一：基层治理（第四章）——特点与功能

2021年真题（新大纲修改）

鸿福社区是一个老旧小区，电动车进楼入户、"飞线充电"屡禁不止，占用公共空间，带来消防安全隐患，邻里纠纷频发，居民对物业和居委会抱怨日益增多。社工小李与居委会合作，走访座谈，发现居民骨干和闲置自行车棚等资源，组织社区议事会进行实质性协商，最终改建充电桩、成立巡逻队，问题显著改善。

【问题1】结合案例，简述基层治理和基层政权建设领域社会工作服务的特点。

【问题2】小李的工作主要发挥了哪些功能？

第一问：特点（双教材对比）

社会版人大版
以社区为对象以村（社区）为主要服务场域
重点解决社区群众面临的共同性问题重点解决基层群众急难愁盼问题
重视社区资源的挖掘和运用重视对基层治理资源的挖掘和运用
强调群众参与，关注人的发展强调协商共治与城乡社区参与
采用宏观结构的视角分析和介入问题—

特点速记：「对象 → 问题 → 资源 → 参与 → 宏观」

① 以社区为场域（对象）
② 解决急难愁盼的集体性问题
③ 资源的挖掘与运用（人力/物力/组织资源）
④ 协商共治，强调群众参与
⑤ 宏观视角系统性解决问题

双版本教材存在差异时：不要求全！最多写5条封顶，前三条两版基本一致，第四条表述不同（群众参与 vs 协商共治），选一个写即可。第五条"宏观视角"能写就写，不会写也OK，属于锦上添花。

案例中如何找对应点？

  • 以社区为对象 → "鸿福社区是一个老旧小区"
  • 共同性/集体性问题 → "大家都反映""经常发生纠纷""安全隐患"
  • 资源挖掘 → 居民骨干（人力资源）、闲置自行车棚（物力资源）、居委会/物业/街道安监办（组织资源）、街道资金支持（财力资源）
  • 协商共治 → "社区议事会""实质性协商""骨干牵头成立巡逻队"
  • 宏观视角 → 小李没有简单"堵"，而是与居委会合作系统性解决

参考答案要点（第一问）：

（1）以城乡社区为主要对象。服务以鸿福社区这一地域性社区为对象，具有多层次性的特点，既包括有充电需求的个体居民，也包括新成立的巡逻队，还涵盖了居委会、物业公司等治理主体。

（2）服务提供主体多元化。案例中的服务是由多方共同提供的，包括专业社工、居委会社区工作者、居民骨干组成的志愿者。

（3）重点解决基层群众的集体性问题。服务聚焦于电动车"飞线充电"问题，这是社区大多数群众共同面临的公共性困境和安全服务空缺。

（4）重视对基层治理资源的挖掘和运用。小李充分挖掘和运用了多种资源：人力资源（居民骨干）、物力资源（闲置自行车棚）、组织资源（居委会、物业、街道安监办、巡逻队）、财力/制度资源（街道资金支持）。

（5）强调协商共治与社区参与。一是发展协商民主，通过组织社区议事会充分听取群众意见，实现了实质性协商。二是强调社区参与，居民围绕共同利益（安全）自下而上地组织起来（成立巡逻队），解决问题，培养了集体意识。

第二问：功能

功能 = 好处！这样做有什么好处？

功能核心关键词速记：

回应需求 → 促进参与 → 链接资源 → 助力调解 → 支持网络 → 促进和谐

【人大版】基层治理的功能（万能背诵）：

  • 回应村（居）民需求，推动公共服务精准化
  • 促进社区参与，增强群众自治能力
  • 链接多元资源，助力构建协同治理格局
  • 助力矛盾调解，维护社区稳定和谐
  • 增能群体发展，培育社区支持网络
  • 助力政策完善，提升治理现代化水平

考试时忘了功能怎么办？

功能 = 好处。把案例中所有问题解决后带来的好处列出来即可：

居民诉求被回应了 → 回应需求
大家一起参与了 → 促进社区参与
链接了各方资源 → 链接资源
纠纷矛盾解决了 → 助力调解
社区更和谐了 → 促进社区和谐

5条小标题 + 案例中提到 + 一句扩写 → 最少8-9分没问题！

基层治理主旋律 = 共建共治共享

社工永远的两个好朋友：链接资源 + 支持网络（考场先写在准考证背面！）

三、案例二：妇女社会工作——需求与参与式学习

2021年真题

社工在某社区调研发现：低收入家庭中有劳动意愿和能力的妇女共30名。她们大多从农村嫁到城里，婚后长期在家照顾老人和孩子，身边几乎没有能谈心的朋友。由于没有工作收入，家庭地位较低；一旦发生家庭矛盾只能忍气吞声。她们希望学一些在家能做的手工编织技能，实现劳动创收和照顾家庭两不误，提升自身价值和家庭地位。

社工组织成立了手工编织互助组。在开展技能培训时，她们认为应由社工制定培训内容和学习计划，因此几乎不发表自己的意见。由于不自信，在讨论环节也很少主动分享自己的想法和经验。

【问题1】分析该社区低收入家庭中的妇女主要有哪几个方面的需求？

【问题2】运用参与式学习方法，社工在开展技能培训时，应采取哪些策略促进这些妇女的改变。

第一问：妇女的需求分析

做题方法：逐句拆解案例，从案例关键词推导需求

案例关键词对应需求
低收入家庭、有劳动意愿增加家庭收入的需求 / 就业的需求
身边几乎没有能谈心的朋友社会交往的需求（构建社会支持网络）
没有工作收入、家庭地位较低、忍气吞声提高家庭收入与家庭地位提升需要（自尊、自信、自我认同）
希望学手工编织技能、实现劳动创收技能提升与自我价值需要
几乎不发表意见、不主动分享自信心提升需要

看到"朋友"→ 想到社会支持网络；看到"地位低""忍气吞声"→ 想到自尊、自信、自我认同、男女平等。这些万能词要放进扩写中，让答案更专业！

参考答案要点（第一问）：

（1）社会交往的需求。案例中社区低收入家庭妇女身边几乎没有能谈心的朋友，表明她们存在社会交往的需求。

（2）提高家庭收入和家庭地位的需求。案例中这些妇女没有工作收入，家庭地位较低，发生家庭矛盾只能忍气吞声，表明她们存在提高家庭收入和家庭地位的需求。

（3）提高技能和自我价值的需求。案例中这些妇女希望学一些在家能做的手工编织技能，实现劳动创收和照顾家庭两不误，提升自身价值和家庭地位。表明她们存在提高技能和自我价值的需求。

（4）提升自信心的需求。案例中这些妇女在参加小组活动时由于自信不足，很少主动分享自己的想法和经验，表明她们存在提升自信的需求。

第二问：参与式学习的策略

参与式学习 = 从"被安排"变成"共同参与者"

核心：她做主、她参与、她决策，全程沉浸式参与各个环节

巧借第一问推策略！

第一问的问题/需求推出的策略
没有朋友 → 社会交往需要促进彼此交流合作
没有工作收入通过技能培训提升能力，帮助就业增收
不发表意见、依赖社工制定课程参与制定培训计划
讨论中很少分享经验鼓励经验分享
习惯被动接受安排建立平等合作关系
缺乏自信、不敢表达增强自信与主体意识

参与式学习三大要点：

（1）使妇女们参与学习课程内容制定、课程主题和进程、课程评估等各个环节，而不是执行专家制定的一套学习计划。

（2）用角色扮演、案例讨论、游戏、情景剧、讲故事等方法调动妇女经验的分享，"小活动、大道理"，通过小活动引发大家的深入思考，让每名妇女充分表达自己的思想。

（3）不仅要让妇女学习到知识，更重要的是提升妇女的自信。社工要通过鼓励发言、肯定进步、创造成功体验等方式，增强妇女的自我认同、自我表达能力和参与动力，推动其从被动接受转向主动参与。

考试时不了解"参与式学习"概念怎么办？

不影响做题！不会的概念不要硬解释。人家问策略直接上策略，从案例中的问题推出来即可。只有第五题方案设计题需要解释理论，案例分析题不需要。

四、案例三：儿童社会工作——需要与家庭寄养

2021年真题

小辉上小学二年级时，爸爸突发急病去世，妈妈因此受刺激得了间歇性精神疾病。妈妈病情发作时不仅没有能力照顾小辉，而且会无故殴打他。最近妈妈病情加重，试图自杀，被邻居及时发现并制止。

村儿童主任紧急报告给镇政府，镇政府委托社工机构入村支援。社工评估发现：小辉今年刚满10岁，身体严重消瘦，已经半个多月没去上学了；小辉的妈妈需要立即入院治疗；小辉家在当地没有亲戚，当前亟需处理小辉的临时安置问题。

上级民政部门决定依法对小辉实施临时监护，委托村委会采取家庭寄养的方式进行临时安置。在征得小辉及其妈妈的知情同意后，面向全村招募临时寄养家庭，并对报名的家庭进行筛选，最终评审出一个合格家庭。

【问题1】结合案例，分析小辉的需要。

【问题2】按照家庭寄养安置介入服务流程，本案例中社工后续需要开展哪些工作？

第一问：儿童的需要

儿童四大需要：「生 发 社 保」

生存的需要 → 发展的需要 → 社会化的需要 → 受保护的需要

看到案例关键词，秒出答案：

  • 父母有一方不在/疾病 → 发展的需要（家庭环境影响身心健康）
  • 无人照顾/身体消瘦 → 生存的需要（生活照料、营养缺乏）
  • 殴打/监护不足/自杀 → 受保护的需要
  • 半个月没上学 → 发展的需要 + 社会化的需要
  • 当地没有亲戚 → 构建社会支持网络的需要（锦上添花）
  • 母亲殴打+自杀 → 心理辅导的需要（锦上添花）

"问题"与"需要"一定要区分清楚！

问问题：用负面词结尾 → "社会支持弱的问题""监护能力不足的问题"

问需要：用正面/中性词 → "构建社会支持网络的需要""受保护的需要"

对于儿童，不要写"小辉面临经济压力"，应写"小辉的家庭面临着较大的经济压力"——主体要严谨！

参考答案要点（第一问）：

（1）生存的需要。案例中提到：小辉的爸爸突发急病去世，妈妈因精神疾病无法正常照顾他，还会无故殴打他，导致小辉"身体严重消瘦"。这说明小辉在生活照顾和基本营养方面缺乏保障，生存状况堪忧。

（2）发展的需要和社会化的需要。案例中提到：小辉已经半个多月没去上学了，明显影响了正常的学习和教育机会，同时没有去上课也影响了他参与学校生活，缺乏与同伴交流的机会。

（3）受保护的需要。案例中提到：妈妈在病情发作时会殴打小辉，还曾试图自杀。这使小辉直接面临人身伤害和心理恐惧的风险。同时，由于爸爸去世、妈妈需入院治疗、家庭缺乏其他监护人，小辉面临严重的家庭监护缺失。

（4）构建社会支持网络的需要。案例中提到，小辉家在当地没有亲戚。说明其正式和非正式支持系统都较为薄弱，需要链接村委会、寄养家庭、学校及相关救助资源，重建支持网络。

（5）心理辅导需要。小辉连续经历丧父、母亲患病、被殴打等多重打击，容易产生恐惧、无助和不安全感，因此需要及时开展心理疏导与情绪支持。

第二问：家庭寄养安置介入流程

家庭寄养六步流程：「招 → 估 → 训 → 配 → 融 → 签」

招募 → 评估 → 培训 → 匹配 → 融合 → 签订协议

案例中已完成：招募 + 评估（筛选出一个合格家庭）

后续需要做的：培训 → 匹配 → 入户融合 → 签订寄养协议

参考答案要点（第二问）：

（1）寄养家庭岗前培训。案例中提到最终评审出一个合格家庭，但在安置前，社工还应对寄养家庭进行培训，帮助其掌握照顾儿童的基本技能，了解儿童保护的法律法规，增强责任意识。

（2）寄养儿童匹配。家庭寄养安置介入包括寄养家庭招募、寄养家庭评估、寄养家庭岗前培训、寄养儿童匹配、儿童入户融合、签订寄养协议。（简述完整流程后聚焦后续步骤）

（3）儿童入户融合。社工需定期家访和电话回访，跟踪小辉在寄养家庭中的生活、学习和情感状态，同时为寄养家庭提供持续的指导和心理支持，解决可能出现的矛盾和问题。
关键词：渐进式融合——从短时接触逐步过渡到长期生活。

（4）签订寄养协议。若小辉与寄养家庭适配并能够顺利融入其中，则组织签订寄养协议。

关于结案/跟进：家庭寄养服务中需要一直跟进，直到寄养关系终止。终止条件包括：妈妈身体恢复健康后小辉回归家庭，或到18岁成年。如果想写结案可以提及，但答案重点是上述四个步骤。

五、案例四：社会救助——社会支持网络

2021年真题

在乡村振兴战略背景下，社工进驻移民搬迁新建社区。通过家访发现：王老汉一家从缺乏基本生存条件的山区搬迁入住新建社区，老伴几年前过世；儿子、儿媳在外省打工，会定期寄钱回家，但人很少回来；孙儿由王老汉独自照顾。王老汉对社区不熟悉，遇到困难不知道该找谁，十分苦闷。

由于搬迁，与王老汉关系密切的原住地乡亲中只剩下十几个人还保持着联系，经常来往的只有两三个。这些乡亲表示，如果王老汉有需要的话，他们愿意提供力所能及的帮助。社工还了解到，新建社区中的移民家庭大多有相互联系的意愿，而且不少热心居民愿意提供志愿服务。

【问题1】分析社工从哪些方面评估了王老汉的社会支持网络状况？

【问题2】社工可以通过哪些具体的步骤强化王老汉的社会支持网络效益？

第一问：社会支持网络评估（四个维度）

社会支持网络评估四维度：

网络规模 → 有多少人
网络基础 → 能帮啥
网络质量 → 能力大不大、意愿强不强
网络强度 → 联系多不多

案例对应分析：

评估维度案例对应结论
网络规模老伴去世、儿子儿媳在外省、经常联系的朋友只有2-3个规模偏小
网络基础儿子儿媳定期寄钱；乡亲愿意提供力所能及的帮助有一定基础，能提供经济支持和生活帮助
网络质量乡亲非常热心、愿意帮助意愿较强，质量尚可
网络强度儿子儿媳"很少回来"、经常来往的只有2-3个联系频率低，强度较弱

考试遇到这种题完全不会怎么保命？

把社会支持网络写成：正式支持网络（政府、机构）+ 非正式支持网络（家人、邻里、朋友、志愿者），然后从案例中提取对应内容，也能拿到5分左右！

第二问：强化社会支持网络效益的步骤

强化支持网络四步走：

旧关系（加强联系）→ 志愿者（链接资源）→ 同类互助（自助互助网络）→ 邻里资源（扩大支持网络）

参考答案要点（第二问）：

（1）协助服务对象巩固原有社会关系。案例中提到：王老汉与原住地乡亲仍保持联系，经常来往的有两三个人，这些乡亲愿意提供帮助。社工可以与王老汉一起梳理现有关系网络，分析哪些成员能够提供支持，并鼓励其加强与乡亲、子女等关系人的联系，巩固原有社会关系，增强社会支持。

（2）链接社区志愿者资源提供支持。案例中提到：社区中有不少热心居民愿意提供志愿服务。社工可以发动社区志愿者资源，将王老汉与愿意提供服务的居民结对联系，通过一对一定期探访等方式，为其提供生活照料和情感支持。

（3）促进形成移民居民互助网络。案例中提到：新建社区中的移民家庭大多有相互联系的意愿。社工可以组织搬迁居民开展交流活动或建立互助小组，促进有相似经历和需求的居民之间相互帮助、相互支持。

（4）动员邻里和社区骨干参与支持。案例中提到：社区中有不少热心居民。社工可以联系社区骨干和邻里，通过邻里互助、社区活动等方式扩大社会支持网络，帮助王老汉更好地融入新的社区生活。

社会支持网络可以和任何一个人群组合出题！妇女、老年人、儿童、残疾人等都可能考。核心逻辑不变：「链接资源 + 支持网络」是永远的老朋友。倡导老龄友好社区建设也可以作为最后的升华句。

六、案例五：方案设计题——小组工作（情绪管理）

2021年真题

某校成立了包括社工在内的学生欺凌防治工作组。社工调查发现，有欺凌行为的学生中不少人缺乏基本的情绪管理能力：有的不能觉察自己的情绪状态；有的不善表达自己的情绪；有的用暴力方式释放负面情绪；有的虽知道欺凌行为不好，但无法自我控制。

在学校的支持下，社工计划运用小组工作方法帮助这些学生科学管理情绪。小组总体目标为：通过提升学生情绪觉察、情绪表达和情绪管理等能力，预防和消除欺凌行为。小组共五节。

【要求】结合案例，完成本情绪管理小组计划的设计，只需列出每节活动的目标和主要内容。

方案设计题万能模板

做题逻辑：问题 → 目标 → 每节的分任务

核心思路：从认知改变 → 行为习惯养成 → 应用于生活

小组活动五节模板（万能框架）：

节次主题核心任务
第一节破冰·认识情绪建立信任关系 + 初步认知
第二节觉察情绪识别自己的情绪反应
第三节表达情绪学会正确表达与宣泄
第四节管理冲突非暴力沟通与冲突应对
第五节巩固能力总结与展望，应用于生活

参考答案：

1. 第一节：认识情绪，建立关系

（1）目标：通过破冰活动让组员互相认识，建立安全氛围；初步掌握情绪的基本知识。

（2）主要内容：①开展暖场游戏，帮助组员消除陌生感，建立小组信任。②社工引导组员围绕"喜、怒、哀、乐"等情绪话题展开讨论，帮助他们认识情绪的性质、种类与作用，以及对行为的影响。

2. 第二节：觉察与识别情绪

（1）目标：引导学生学会觉察并识别自己的情绪状态。

（2）主要内容：①通过情绪量表测评，帮助组员认识自己常见的情绪反应。②讨论生活中不良情绪的表现，如愤怒、焦虑、失落等，引导学生联系实际，觉察自己在不同情境下的情绪反应。

3. 第三节：正确表达与宣泄情绪

（1）目标：帮助学生学会表达情绪，并掌握健康的情绪宣泄方式。

（2）主要内容：①角色扮演练习，引导组员用语言和合适的方式表达愤怒、兴奋、紧张等情绪，而不是以暴力方式释放。②学习转移注意力的方法，如运动、音乐、写日记等，掌握积极的情绪宣泄技巧。③小组分享，鼓励组员总结自己最喜欢和最适合的情绪管理方式。

4. 第四节：非暴力沟通与冲突应对

（1）目标：提升学生的沟通能力和情绪控制能力，减少因沟通不当引发的冲突。

（2）主要内容：①社工讲解"非暴力沟通"的四要素（观察、感受、需要、请求）。②小组练习：结合生活中可能出现的同伴冲突情境，分组模拟对话，学习用平和方式表达需求。③讨论和反思：通过交流体验，掌握控制情绪、防止冲突升级的技巧。

5. 第五节：总结与展望

（1）目标：巩固学习成果，总结小组经验，处理活动结束带来的情绪，展望未来。

（2）主要内容：①邀请组员分享小组中的收获与变化，表达对他人的感谢。②共同制定"情绪管理小锦囊"，总结小组中学到的技巧，作为今后生活中的参考。③社工引导组员展望未来，强调"成为情绪的主人"，并处理好小组离别带来的情绪体验。

方案设计题是最好提分的一道题！目标12-14分。

模板性极强，记住框架直接套用。如果考试是六节，增加一节内容；如果是五节，删减一节即可。

考试技巧：方案设计题建议直接抄模板，约800字左右。字大的写700字。不要写太满，也不要跳行写。

方案设计题可能考的主题：情绪管理（学校/青少年）、人在情境中理论、生态系统理论、优势视角理论等。把"情绪"换成其他主题，第一步破冰、最后一步巩固应用于生活的框架不变！
